HTML Tags and Attributes

Welcome to our HTML learning page! In this lesson, we'll cover the basics of HTML tags and attributes.

HTML Tags

HTML uses tags to structure content. Here are some essential tags:

  • <html>: The root element
  • <head>: Contains meta-information
  • <body>: Holds the content

HTML Attributes

Attributes provide additional information about HTML elements. Common attributes include:

  • class: Specifies a class for styling
  • id: Identifies a unique element
  • src: Defines the source of an element (e.g., image)
